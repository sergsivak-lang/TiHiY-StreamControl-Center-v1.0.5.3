DONATELLO API
1. Відкрийте Donatello → API.
2. Увімкніть API та скопіюйте Token.
3. У програмі відкрийте МОДУЛІ → DONATELLO / ДОНАТИ.
4. Вставте Token у поле API Token (X-Token).
5. Натисніть ЗБЕРЕГТИ → ПЕРЕВІРИТИ API TOKEN → ПІДКЛЮЧИТИ.
6. Не генеруйте новий Donatello token без потреби: це може змінити URL віджетів Donatello.

Програма використовує:
GET https://donatello.to/api/v1/me
GET https://donatello.to/api/v1/donates?page=0&size=20
GET https://donatello.to/api/v1/subscribers?isActive=true&page=0&size=20
Header: X-Token: <ваш token>

DISCORD
У модулі СПОВІЩЕННЯ / DISCORD BOT є два незалежні блоки:
- ID каналів початку трансляцій;
- ID каналів донатів і платних підписок.

Після заповнення натисніть:
- ТЕСТ КАНАЛУ СТРІМІВ;
- ТЕСТ КАНАЛУ ДОНАТІВ.

OBS ALERTS
1. Відкрийте МОДУЛІ → OVERLAY.
2. Скопіюйте Alerts overlay URL.
3. В OBS додайте Browser Source розміром приблизно 1100×500.
4. Джерело має бути прозорим; новий донат або підписка з'являється автоматично.

TWITCH / YOUTUBE
Авторизації залишаються в модулі КАНАЛИ.
Donatello може окремо надсилати коротке повідомлення у підключені Twitch/YouTube чати.
