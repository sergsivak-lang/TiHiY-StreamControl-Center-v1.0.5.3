TiHiY StreamControl Center — підключення AIDA64
================================================

Блок «МОНІТОРИНГ ПК» показує реальні показники:
- поточний час;
- частоту CPU;
- частоту оперативної пам’яті;
- завантаження CPU і GPU;
- температуру CPU і GPU;
- завантаження RAM і VRAM.

Для температур, частот, GPU та VRAM програма читає AIDA64 Shared Memory:
AIDA64_SensorValues.

1. Запустіть AIDA64.
2. Відкрийте Налаштування / Preferences.
3. Перейдіть до External Applications / Зовнішні програми.
4. Увімкніть Shared Memory / Спільна пам’ять.
5. У списку сенсорів залиште доступними CPU Clock, Memory Clock, CPU/GPU Utilization, CPU/GPU Temperature, Memory Utilization і GPU Memory Utilization.
6. Не закривайте AIDA64 під час роботи TiHiY StreamControl Center.

У блоці з’явиться зелений статус AIDA64 LIVE.

Якщо AIDA64 не запущена або Shared Memory вимкнена:
- CPU load, RAM і мережа беруться з Windows;
- температури, частоти, GPU та VRAM показуються як «—»;
- вигадані значення не підставляються.

Розміри блоків
---------------
- перетягуйте світні розділювачі мишкою;
- вертикальні розділювачі змінюють ширину;
- горизонтальні — висоту;
- розміри автоматично зберігаються;
- подвійний клік на розділювачі повертає стандартний макет.

Доступність конкретних датчиків залежить від обладнання, драйверів
і того, які сенсори бачить AIDA64 на цьому ПК.
