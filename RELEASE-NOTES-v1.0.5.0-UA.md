# TiHiY StreamControl Center v1.0.5.2 — Ukraine Approved Visual Theme

Ця версія продовжує роботу над однією еталонною темою «Україна».

## Змінено
- прибрано суцільний орнамент із верхньої частини кожного блока;
- орнамент залишено як стриманий кутовий/верхній акцент;
- створено новий темний синьо-графітовий фон без декоративного шуму;
- створено нову слабку текстуру панелей без яскравої смуги;
- панелі отримали подвійні cyan/золоті внутрішні рамки;
- зменшено прозорість тризубів і орнаментів;
- палітру теми зміщено ближче до затвердженого ескізу: темно-синя база, cyan-заголовки, золоті акценти;
- архів очищено від bin, obj, .vs, BuildLogs і тимчасових файлів.

## Перевірка
Статично перевірено 14 XAML-файлів. Фактичне WPF-складання та відображення перевіряються на Windows.
