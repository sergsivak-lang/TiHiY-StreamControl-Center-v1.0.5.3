TiHiY StreamControl Center — погоджена тема STALKER v4

Цей пакет виправляє помилку v3 і працює з реальною grid-структурою фінального MainWindow.xaml.

Графіка:
- жодного нового намальованого або згенерованого оформлення;
- усі рамки, болти, шапка, шеврон, панорама, hazard-смуги та центральний банер походять із погодженого ескізу 1672×941;
- динамічні приклади тексту прибрані з оболонок тільки точним panel-fill-dark.png з того самого ескізу.

Код:
- використовує DesignSurface, DashboardBlocksGrid, TopBlocksGrid, BottomBlocksGrid і FooterBlocksGrid;
- вимикає попередні плоскі StalkerThemeRuntime, StalkerThemeFinalizer і StalkerSettingsRuntime, щоб їхні таймери не перезаписували точну тему;
- залишає живими чат, донати, мікшер, сповіщення, системні показники й AIDA64;
- тема «Україна» відновлюється при перемиканні назад.

Фінальне прийняття можливе лише після Windows Release Build, запуску EXE та порівняння реального скріну з ескізом.
